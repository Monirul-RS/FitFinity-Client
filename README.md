## Features

- Light/dark mode toggle
- Service Details 
- Premium Access Available
- Fullscreen mode
- Cross platform
- Google, Github and Email Sign in.
- Review Options


## Tech Stack

**Client:** React, tailwindCss, DaisyUi, FontAwesome, Toastify

**Server:** Node, Express, Mongodb

`API_KEY`

`ANOTHER_API_KEY`
